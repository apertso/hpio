# Spacing

The design system uses the Tailwind CSS default spacing scale. The base unit is **0.25rem (4px)**.

## Usage Guidelines

- **Layout Padding:** Use `p-4` (16px) or `p-6` (24px) for containers.
- **Component Spacing:** Use `gap-2` (8px) to `gap-4` (16px) for grouping related elements.
- **Section Spacing:** Use `space-y-6` (24px) or `space-y-8` (32px) between major page sections.
