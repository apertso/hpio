# Typography

The application uses **Roboto** as the primary font family, falling back to system sans-serif fonts.

**Font Family:** `"Roboto", "Segoe UI", "Tahoma", "Arial", "Helvetica", sans-serif`

- **Regular (400):** Default text
- **Medium (500):** Labels, buttons, subheaders
- **Bold (700):** Headers, emphasis
